Common Questions

Is CAD Layer Map Builder free?

Yes. This version is a free static tool with AdSense and browser-side project storage.

Does it send my inputs to a server?

No. Inputs and saved projects stay in the browser for this static build.

Can I use the output professionally?

Use it as a draft or review aid. Final use still needs qualified professional judgment.
